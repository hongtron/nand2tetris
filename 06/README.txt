Function:
This program is designed to translate from Hack assembly language into machine language.

Requirements:
Python 2.7

Running:
To run the program, first navigate to the project directory (/06/) via the command line.
The syntax to run the program is as follows.

python assembler/assembler.py /path/to/input/file.asm

Input/Output:
The input file should be a .asm file. The output file will use the same filename with the extension .hack, and will be placed in the same directory as the input file.
